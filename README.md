# Food Delivery App - ElifTech School Test Task

### 🚀 Accomplished Level: **Middle Level**

This application allows users to browse various shops, filter products, and manage a shopping cart for food delivery.

## 🛠 Features (Middle Level Implementation)
- [cite_start]**Responsive Design**: Fully functional on Desktop, Tablet, and Mobile devices.
- [cite_start]**Shop Filtering by Rating**: Users can filter shops based on their rating ranges (e.g., 4.0-5.0).
- [cite_start]**Product Filtering by Category**: Products can be filtered by specific categories (Burgers, Drinks, etc.).
- [cite_start]**Product Sorting**: Dynamic sorting by price (Asc/Desc) and Name (A-Z).
- [cite_start]**Persistent Shopping Cart**: Items are saved in LocalStorage and synchronized with the database on submission.
- [cite_start]**Form Validation**: Comprehensive validation for user information before order submission.

## 💻 Tech Stack
- [cite_start]**Frontend**: Next.js (App Router), Tailwind CSS, shadcn/ui.
- [cite_start]**Backend**: Node.js, Prisma ORM.
- [cite_start]**Database**: PostgreSQL.
